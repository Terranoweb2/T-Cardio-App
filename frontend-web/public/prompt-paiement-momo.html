1. Presentation du Systeme de Paiement

Implemente un systeme de paiement Mobile Money (MoMo) complet pour une application de telemedecine en Afrique de l'Ouest francophone. Le systeme gere les abonnements, les achats de credits de consultation, et les paiements via plusieurs operateurs Mobile Money.

Objectifs

  • Supporter CinetPay comme aggregateur principal (Orange Money, MTN MoMo, Wave, Moov, Free Money)
  • Supporter PayDunya comme aggregateur secondaire/fallback
  • Conserver le mode MoMo Local USSD (paiement manuel avec confirmation admin)
  • Gerer les abonnements (plans mensuels/annuels)
  • Gerer les credits de consultation (achat de packs)
  • Gerer les remboursements et decaissements (payout vers operateurs)
  • Dashboard admin pour gestion et suivi des transactions
  • Webhooks securises pour confirmation automatique

2. Architecture du Module de Paiement

Stack Technique

  • Backend : NestJS (TypeScript), Prisma ORM, PostgreSQL
  • Frontend : Next.js 14 (App Router), TailwindCSS
  • Cache : Redis (pour verrouillage de transactions et cache de tokens)
  • Queue : Bull/BullMQ (pour traitement asynchrone des webhooks)
  • Notifications : WebSocket (Socket.io) + Push notifications

  readonly name: string;

  initiatePayment(params: InitiateParams): Promise<InitiateResult>;
  verifyPayment(transactionId: string): Promise<VerifyResult>;

Provider CinetPay

// providers/cinetpay.provider.ts
import { Injectable, Logger } from '@nestjs/common';
import { ConfigService } from '@nestjs/config';
import axios from 'axios';
import * as crypto from 'crypto';
import { PaymentProvider, InitiateParams, InitiateResult,
         VerifyResult, WebhookResult } from './payment-provider.interface';

@Injectable()
export class CinetPayProvider implements PaymentProvider {
  readonly name = 'CINETPAY';
  private readonly logger = new Logger(CinetPayProvider.name);
  private readonly apiKey: string;
  private readonly siteId: string;
  private readonly secretKey: string;
  private readonly baseUrl: string;
  private readonly notifyUrl: string;
  private readonly returnUrl: string;

  constructor(private config: ConfigService) {
    this.apiKey = this.config.get('cinetpay.apiKey');
    this.siteId = this.config.get('cinetpay.siteId');
    this.secretKey = this.config.get('cinetpay.secretKey');
    this.baseUrl = this.config.get('cinetpay.baseUrl');
    this.notifyUrl = this.config.get('cinetpay.notifyUrl');
    this.returnUrl = this.config.get('cinetpay.returnUrl');
  }

  // ---- Initier un paiement ----
  async initiatePayment(params: InitiateParams): Promise<InitiateResult> {
    try {
      const payload = {
        apikey: this.apiKey,
        site_id: this.siteId,
        transaction_id: params.transactionId,
        amount: params.amount,
        currency: params.currency,
        description: params.description,
        notify_url: params.notifyUrl || this.notifyUrl,
        return_url: params.returnUrl || this.returnUrl,
        channels: params.channels || 'MOBILE_MONEY',
        lang: 'fr',
        // Metadata client
        customer_name: params.customerName || '',
        customer_surname: '',
        customer_email: params.customerEmail || '',
        customer_phone_number: params.customerPhone || '',
        customer_address: '',
        customer_city: '',
        customer_country: params.metadata?.country || 'CI',
        customer_state: '',
        customer_zip_code: '',
        // Metadata custom
        metadata: JSON.stringify(params.metadata || {}),
      };

      const { data } = await axios.post(
        `${this.baseUrl}/payment`, payload
      );

      if (data.code === '201') {
        return {
          success: true,
          providerTransactionId: params.transactionId,
          paymentUrl: data.data.payment_url,
          expiresAt: new Date(Date.now() + 30 * 60 * 1000), // 30 min
        };
      }

      this.logger.error(`CinetPay initiation failed: ${JSON.stringify(data)}`);
      return { success: false, providerTransactionId: '' };
    } catch (error) {
      this.logger.error(`CinetPay error: ${error.message}`);
      throw error;
    }
  }

  // ---- Verifier un paiement ----
  async verifyPayment(transactionId: string): Promise<VerifyResult> {
    const { data } = await axios.post(
      `${this.baseUrl}/payment/check`,
      {
        apikey: this.apiKey,
        site_id: this.siteId,
        transaction_id: transactionId,
      }
    );

    const statusMap: Record<string, VerifyResult['status']> = {
      'ACCEPTED': 'COMPLETED',
      'REFUSED': 'FAILED',
      'CANCELLED': 'CANCELLED',
    };

    return {
      status: statusMap[data.data?.status] || 'PENDING',
      providerTransactionId: transactionId,
      amount: parseInt(data.data?.amount || '0'),
      currency: data.data?.currency || 'XOF',
      paymentMethod: data.data?.payment_method,  // OM, MOMO, WAVE, etc.
      operatorTransactionId: data.data?.operator_id,
      paidAt: data.data?.payment_date ? new Date(data.data.payment_date) : undefined,
      rawData: data,
    };
  }

  // ---- Traiter le webhook ----
  async handleWebhook(payload: any, headers: Record<string, string>): Promise<WebhookResult> {
    // Verifier la signature (x-token header)
    // CinetPay envoie un POST avec cpm_trans_id
    const transactionId = payload.cpm_trans_id;

    if (!transactionId) {
      throw new Error('Missing transaction ID in webhook');
    }

    // TOUJOURS re-verifier aupres de CinetPay (ne jamais faire confiance au webhook seul)
    const verification = await this.verifyPayment(transactionId);

    return {
      verified: true,
      transactionId,
      status: verification.status,
      amount: verification.amount,
      paymentMethod: verification.paymentMethod,
      operatorTransactionId: verification.operatorTransactionId,
    };
  }
}

Flux de Paiement CinetPay

1
Patient clique "Payer" - Le frontend envoie POST /api/v1/payments/initiate avec { type, packageId }
2
Backend cree Payment - Statut PENDING, genere un transactionId unique, appelle CinetPay /v2/payment
3
CinetPay retourne payment_url - Le frontend redirige le patient vers la page de checkout CinetPay
4
Patient selectionne operateur - Orange Money, MTN, Wave, etc. et valide sur son telephone
5
CinetPay envoie webhook - POST vers notify_url avec cpm_trans_id
6
Backend verifie - Appelle /v2/payment/check pour confirmer le statut (JAMAIS faire confiance au webhook seul)
7
Activation - Si COMPLETED : active l'abonnement OU ajoute les credits, notifie le patient via WebSocket

@Injectable()
export class PayDunyaProvider implements PaymentProvider {
  readonly name = 'PAYDUNYA';
  private readonly logger = new Logger(PayDunyaProvider.name);

  constructor(private config: ConfigService) {}

  private getHeaders() {
    return {
      'PAYDUNYA-MASTER-KEY': this.config.get('paydunya.masterKey'),
      'PAYDUNYA-PRIVATE-KEY': this.config.get('paydunya.privateKey'),
      'PAYDUNYA-TOKEN': this.config.get('paydunya.token'),
      'Content-Type': 'application/json',
    };
  }

  async initiatePayment(params: InitiateParams): Promise<InitiateResult> {
    const baseUrl = this.config.get('paydunya.baseUrl');
    const payload = {
      invoice: {
        total_amount: params.amount,
        description: params.description,
      },
      store: {
        name: 'T-Cardio Plus',    // ou T-Gyneco Plus
        tagline: 'Telemedecine',
      },
      actions: {
        cancel_url: `${this.config.get('app.frontendUrl')}/payment/cancel`,
        return_url: `${this.config.get('app.frontendUrl')}/payment/success`,
        callback_url: params.notifyUrl || this.config.get('paydunya.callbackUrl'),
      },
      custom_data: {
        transaction_id: params.transactionId,
        ...params.metadata,
      },
    };

    const { data } = await axios.post(
      `${baseUrl}/checkout-invoice/create`,
      payload,
      { headers: this.getHeaders() }
    );

    if (data.response_code === '00') {
      return {
        success: true,
        providerTransactionId: data.token,
        paymentUrl: data.response_text,      // URL de checkout
      };
    }

    return { success: false, providerTransactionId: '' };
  }

  async verifyPayment(invoiceToken: string): Promise<VerifyResult> {
    const baseUrl = this.config.get('paydunya.baseUrl');
    const { data } = await axios.get(
      `${baseUrl}/checkout-invoice/confirm/${invoiceToken}`,
      { headers: this.getHeaders() }
    );

    const statusMap: Record<string, VerifyResult['status']> = {
      'completed': 'COMPLETED',
      'pending': 'PENDING',
      'cancelled': 'CANCELLED',
      'failed': 'FAILED',
    };

    return {
      status: statusMap[data.status] || 'PENDING',
      providerTransactionId: invoiceToken,
      amount: data.invoice?.total_amount || 0,
      currency: 'XOF',
      paymentMethod: data.customer?.payment_method,
      rawData: data,
    };
  }

  async handleWebhook(payload: any, headers: Record<string, string>): Promise<WebhookResult> {
    // PayDunya envoie data en x-www-form-urlencoded
    // Inclut un hash SHA-512 pour verification
    const transactionId = payload.custom_data?.transaction_id;

    // Verifier la signature avec le master key
    // hash = SHA512(masterKey)

    const verification = await this.verifyPayment(payload.token);

    return {
      verified: true,
      transactionId,
      status: verification.status,
      amount: verification.amount,
      paymentMethod: verification.paymentMethod,
    };
  }

// providers/momo-local.provider.ts
@Injectable()
export class MomoLocalProvider implements PaymentProvider {
  readonly name = 'MOMO_LOCAL';
  private readonly receiverNumber: string;

  constructor(private config: ConfigService) {
    this.receiverNumber = this.config.get('payment.momoReceiverNumber');
  }

  async initiatePayment(params: InitiateParams): Promise<InitiateResult> {
    const reference = this.generateReference();
    const ussdCode = this.generateUssdCode(
      params.amount, reference, params.metadata?.country
    );

    return {
      success: true,
      providerTransactionId: reference,
      ussdCode,
    };
  }

  // Generer reference unique
  private generateReference(): string {
    const ts = Date.now().toString(36).toUpperCase();
    const rand = Math.random().toString(36).substring(2, 6).toUpperCase();
    return `TC-${ts}-${rand}`;
  }

  // Generer code USSD selon l'operateur et le pays
  private generateUssdCode(amount: number, ref: string, country?: string): string {
    const receiver = this.receiverNumber;

    // Formats USSD par operateur (exemples)
    const formats: Record<string, string> = {
      // Orange Money CI
      'OM_CI': `#144*1*1*${receiver}*${amount}*${ref}#`,
      // MTN CI
      'MTN_CI': `*133*1*1*${receiver}*${amount}*${ref}#`,
      // Wave
      'WAVE': `Ouvrir l'app Wave > Envoyer ${amount} XOF a ${receiver} ref: ${ref}`,
      // Moov
      'MOOV_CI': `*155*1*1*${receiver}*${amount}*${ref}#`,
      // Default
      'DEFAULT': `*880*1*1*${receiver}*${receiver}*${amount}*${ref}*PIN#`,
    };

    return formats[country] || formats['DEFAULT'];
  }

  // Pas de verification auto pour MoMo local
  async verifyPayment(transactionId: string): Promise<VerifyResult> {
    return {
      status: 'PENDING',  // Toujours en attente de confirmation admin
      providerTransactionId: transactionId,
      amount: 0,
      currency: 'XOF',
    };
  }

  async handleWebhook(): Promise<WebhookResult> {
    throw new Error('MoMo local ne supporte pas les webhooks');
  }
}

@Injectable()
export class WebhookSignatureGuard implements CanActivate {
  constructor(private config: ConfigService) {}

  canActivate(context: ExecutionContext): boolean {
    const request = context.switchToHttp().getRequest();
    const provider = request.params.provider;  // cinetpay ou paydunya

    switch (provider) {
      case 'cinetpay':
        return this.verifyCinetPay(request);
      case 'paydunya':
        return this.verifyPayDunya(request);
      default:
        return false;
    }
  }

  private verifyCinetPay(request: any): boolean {
    // CinetPay envoie un header x-token
    // Verifier que l'IP source est dans les ranges CinetPay
    const allowedIPs = [
      '196.28.244.0/24',    // Range CinetPay
      '41.203.0.0/16',      // Range alternatif
    ];
    // En production, verifier l'IP source
    return true;  // Toujours re-verifier via /check
  }

  private verifyPayDunya(request: any): boolean {
    // PayDunya: verifier le hash SHA-512
    const masterKey = this.config.get('paydunya.masterKey');
    const expectedHash = crypto
      .createHash('sha512')
      .update(masterKey)
      .digest('hex');

    return request.headers['paydunya-hash'] === expectedHash;
  }

import { Processor, Process } from '@nestjs/bull';
import { Job } from 'bull';

@Processor('payment-webhooks')
export class WebhookProcessor {
  constructor(
    private paymentService: PaymentService,
    private logger: Logger,
  ) {}

  @Process('process-webhook')
  async handleWebhook(job: Job<{
    provider: string;
    payload: any;
    headers: Record<string, string>;
    webhookEventId: string;
  }>) {
    const { provider, payload, headers, webhookEventId } = job.data;

    try {
      // 1. Recuperer le bon provider
      const paymentProvider = this.paymentService.getProvider(provider);

      // 2. Traiter le webhook (inclut la re-verification)
      const result = await paymentProvider.handleWebhook(payload, headers);

      // 3. Mettre a jour le paiement interne
      if (result.verified) {
        await this.paymentService.processWebhookResult(result);
      }

      // 4. Marquer l'evenement webhook comme traite
      await this.paymentService.updateWebhookEvent(webhookEventId, 'PROCESSED');

    } catch (error) {
      this.logger.error(`Webhook processing failed: ${error.message}`);
      await this.paymentService.updateWebhookEvent(webhookEventId, 'FAILED', error.message);
      throw error;  // Bull retente automatiquement
    }
  }
}

Idempotence

// Dans PaymentService
async processWebhookResult(result: WebhookResult): Promise<void> {
  // Verrou Redis pour eviter le traitement concurrent
  const lockKey = `payment:webhook:${result.transactionId}`;
  const lock = await this.redis.set(lockKey, '1', 'EX', 60, 'NX');

  if (!lock) {
    this.logger.warn(`Webhook already processing: ${result.transactionId}`);
    return;  // Un autre worker traite deja ce webhook
  }

  try {
    const payment = await this.prisma.payment.findFirst({
      where: {
        OR: [
          { id: result.transactionId },
          { providerTransactionId: result.transactionId },
          { momoReference: result.transactionId },
        ]
      },
    });

    if (!payment) throw new Error(`Payment not found: ${result.transactionId}`);

    // Idempotence: ignorer si deja traite
    if (payment.status === 'COMPLETED' || payment.status === 'REFUNDED') {
      this.logger.log(`Payment already ${payment.status}, ignoring webhook`);
      return;
    }

    // Transition de statut
    if (result.status === 'COMPLETED') {
      await this.completePayment(payment, result);
    } else if (result.status === 'FAILED' || result.status === 'CANCELLED') {
      await this.failPayment(payment, result);
    } else if (result.status === 'REFUNDED') {
      await this.refundPayment(payment, result);
    }
  } finally {
    await this.redis.del(lockKey);
  }
}

private async completePayment(payment: Payment, result: WebhookResult) {
  await this.prisma.$transaction(async (tx) => {
    // 1. Mettre a jour le paiement
    await tx.payment.update({
      where: { id: payment.id },
      data: {
        status: 'COMPLETED',
        channel: this.mapChannel(result.paymentMethod),
        operatorTransactionId: result.operatorTransactionId,
        completedAt: new Date(),
        netAmount: result.amount,
      },
    });

    // 2. Logger l'evenement
    await tx.paymentEvent.create({
      data: {
        paymentId: payment.id,
        eventType: 'PAYMENT_COMPLETED',
        fromStatus: payment.status,
        toStatus: 'COMPLETED',
        provider: payment.provider,
        triggeredBy: 'WEBHOOK',
      },
    });

    // 3. Activer abonnement ou ajouter credits
    const metadata = payment.metadata as any;
    if (payment.type === 'SUBSCRIPTION') {
      await this.subscriptionService.activateSubscription(
        payment.patientId, metadata.planId, payment.id, payment.amount
      );
    } else if (payment.type === 'CREDIT_PURCHASE') {
      const pkg = this.getPackage(metadata.packageId);
      await this.creditService.addCredits(
        payment.patientId, pkg.credits + pkg.bonus, payment.id,
        `Achat pack ${pkg.name}`
      );
    }

    // 4. Notifier le patient en temps reel
    this.notificationGateway.sendToUser(payment.patientId, 'payment:completed', {
      paymentId: payment.id,
      amount: payment.amount,
      type: payment.type,
    });
  });
}

import { Injectable } from '@nestjs/common';

@Injectable()
export class SubscriptionScheduler {
  constructor(
    private subscriptionService: SubscriptionService,
    private paymentService: PaymentService,
    private notificationService: NotificationService,
  ) {}

  // Toutes les heures: expirer les abonnements passes
  @Cron(CronExpression.EVERY_HOUR)
  async expireOverdueSubscriptions() {
    const expired = await this.subscriptionService.expireOverdueSubscriptions();
    this.logger.log(`Expired ${expired} subscriptions`);
  }

  // Chaque jour a 9h: rappel d'expiration (7 jours avant)
  @Cron('0 9 * * *')
  async sendExpirationReminders() {
    const expiring = await this.subscriptionService.findExpiringSubscriptions(7);
    for (const sub of expiring) {
      await this.notificationService.send(sub.patientId, {
        title: 'Abonnement expire bientot',
        body: `Votre abonnement ${sub.plan} expire dans ${this.daysUntil(sub.endDate)} jours.`,
        type: 'SUBSCRIPTION_EXPIRING',
      });
    }
  }

  // Chaque jour a 10h: rappel 3 jours avant
  @Cron('0 10 * * *')
  async sendUrgentReminders() {
    const expiring = await this.subscriptionService.findExpiringSubscriptions(3);
    for (const sub of expiring) {
      await this.notificationService.send(sub.patientId, {
        title: 'Abonnement expire dans 3 jours !',
        body: 'Renouvelez votre abonnement pour continuer a profiter de tous les services.',
        type: 'SUBSCRIPTION_URGENT',
        actionUrl: '/payment',
      });
    }
  }

  // Chaque jour a minuit: expirer les paiements MoMo locaux non confirmes (24h)
  @Cron(CronExpression.EVERY_DAY_AT_MIDNIGHT)
  async expireStaleMomoPayments() {
    const cutoff = new Date(Date.now() - 24 * 60 * 60 * 1000);
    const expired = await this.paymentService.expirePayments({
      provider: 'MOMO_LOCAL',
      status: 'PENDING',
      createdBefore: cutoff,
    });
    this.logger.log(`Expired ${expired} stale MoMo payments`);
  }

  // Toutes les 5 minutes: expirer les sessions CinetPay/PayDunya (30 min)
  @Cron('*/5 * * * *')
  async expirePaymentSessions() {
    const cutoff = new Date(Date.now() - 30 * 60 * 1000);
    const expired = await this.paymentService.expirePayments({
      provider: ['CINETPAY', 'PAYDUNYA'],
      status: 'PENDING',
      createdBefore: cutoff,
    });
    if (expired > 0) {
      this.logger.log(`Expired ${expired} payment sessions`);
    }
  }

  // Chaque semaine: rapport financier
  @Cron('0 8 * * 1')  // Lundi a 8h
  async generateWeeklyReport() {
    await this.paymentService.generateWeeklyReport();
  }

  // Chaque mois: expirer les credits non utilises (si applicable)
  @Cron('0 0 1 * *')  // 1er du mois a minuit
  async expireOldCredits() {
    const expired = await this.creditService.expireCredits(365); // Credits > 1 an

  }

Service de Decaissement

// payout/payout.service.ts
@Injectable()
export class PayoutService {

  // Calculer les gains d'un medecin
  async getDoctorEarnings(doctorId: string, periodStart?: Date, periodEnd?: Date) {
    const consultations = await this.prisma.teleconsultation.findMany({
      where: {
        doctorId,
        status: 'COMPLETED',
        completedAt: {
          gte: periodStart || new Date(0),
          lte: periodEnd || new Date(),
        },
        // Exclure les consultations deja decaissees
        payoutId: null,
      },
    });

    const totalEarnings = consultations.reduce((sum, c) => {
      const amount = c.type === 'EMERGENCY' ? 1000 : 5000;
      return sum + Math.floor(amount * 0.70);  // 70% medecin
    }, 0);

    return {
      doctorId,
      consultationCount: consultations.length,
      grossAmount: consultations.reduce((s, c) => s + (c.type === 'EMERGENCY' ? 1000 : 5000), 0),
      doctorShare: totalEarnings,
      platformShare: consultations.reduce((s, c) => {
        const amount = c.type === 'EMERGENCY' ? 1000 : 5000;
        return s + Math.floor(amount * 0.30);
      }, 0),
      consultationIds: consultations.map(c => c.id),
    };
  }

  // Initier un decaissement
  async initiatePayout(
    adminId: string,
    doctorId: string,
    recipientPhone: string,
    withdrawMode: string,   // orange-money-senegal, wave, etc.
  ) {
    const earnings = await this.getDoctorEarnings(doctorId);

    if (earnings.doctorShare < payoutConfig.minimumPayoutAmount) {
      throw new BadRequestException(
        `Montant insuffisant: ${earnings.doctorShare} XOF (minimum: ${payoutConfig.minimumPayoutAmount} XOF)`
      );
    }

    // Creer le payout en base
    const payout = await this.prisma.payout.create({
      data: {
        recipientId: doctorId,
        recipientType: 'DOCTOR',
        recipientPhone,
        recipientName: earnings.doctorName,
        amount: earnings.doctorShare,
        currency: 'XOF',
        feeAmount: 0,
        netAmount: earnings.doctorShare,
        provider: 'PAYDUNYA',
        withdrawMode,
        status: 'PENDING',
        description: `Paiement ${earnings.consultationCount} consultations`,
        referenceIds: earnings.consultationIds,
        periodStart: earnings.periodStart,
        periodEnd: new Date(),
        initiatedBy: adminId,
      },
    });

    return payout;
  }

  // Executer le decaissement via PayDunya
  async executePayout(payoutId: string, approvedBy: string) {
    const payout = await this.prisma.payout.findUniqueOrThrow({
      where: { id: payoutId },
    });

    const provider = this.paymentService.getProvider('PAYDUNYA') as PayDunyaProvider;

    const result = await provider.disburseFunds({
      amount: payout.amount,
      currency: payout.currency,
      recipientPhone: payout.recipientPhone,
      recipientName: payout.recipientName,
      withdrawMode: payout.withdrawMode,
      description: payout.description,
      transactionId: payout.id,
    });

    // Mettre a jour le payout
    await this.prisma.payout.update({
      where: { id: payoutId },
      data: {
        status: result.success ? 'COMPLETED' : 'FAILED',
        providerPayoutId: result.refundId,
        providerRawData: result as any,
        approvedBy,
        processedAt: result.success ? new Date() : undefined,
        errorMessage: result.success ? undefined : result.status,
      },
    });

    // Marquer les consultations comme decaissees
    if (result.success) {
      await this.prisma.teleconsultation.updateMany({
        where: { id: { in: payout.referenceIds } },
        data: { payoutId: payout.id },
      });
    }

    return result;
  }
}

15. Securite et Bonnes Pratiques

REGLES CRITIQUES DE SECURITE

Securite des Paiements

  • JAMAIS stocker les cles API dans le code source - Toujours utiliser les variables d'environnement
  • TOUJOURS re-verifier les webhooks - Ne jamais faire confiance aux donnees du webhook seules, appeler /check ou /confirm
  • Verifier la signature - Utiliser le header x-token (CinetPay) ou SHA-512 hash (PayDunya)
  • Idempotence - Gerer les webhooks dupliques (verrou Redis + check status)
  • Transactions atomiques - Utiliser Prisma.$transaction pour paiement + activation
  • Logger TOUS les evenements - Table PaymentEvent pour audit complet
  • Rate limiting - Proteger les endpoints de paiement (sauf webhooks)
  • HTTPS obligatoire - Tous les webhooks et callbacks en HTTPS

Prevention de la Fraude

  • Verifier le montant - Comparer le montant webhook vs montant attendu en base
  • Limiter les tentatives - Maximum 3 paiements en attente par patient
  • Expiration automatique - Sessions CinetPay/PayDunya: 30 min, MoMo local: 24h
  • Reference unique - Chaque transaction a un ID unique non predictible
  • Double validation MoMo local - Verification admin obligatoire
  • IP whitelisting webhooks - En production, filtrer par IP source du provider

async initiateWithFallback(params: InitiateParams): Promise<InitiateResult> {
  try {
    const primary = this.getProvider(this.config.get('defaultProvider'));
    return await primary.initiatePayment(params);
  } catch (error) {
    this.logger.warn(`Primary provider failed, falling back: ${error.message}`);
