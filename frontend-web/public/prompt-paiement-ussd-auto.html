Paiement USSD Automatique

Prompt Claude Code - Systeme de paiement MoMo automatique en arriere-plan

Le patient entre seulement son PIN MoMo. L'appel USSD est invisible et automatique. Aucun numero affiche.

1. Objectif Principal

COMPORTEMENT ATTENDU : Quand le patient clique "Payer", une popup apparait et demande UNIQUEMENT son code PIN MoMo (4 ou 5 chiffres). Il valide. L'appel USSD se fait automatiquement EN ARRIERE-PLAN. Le patient ne voit JAMAIS le numero USSD, ne voit JAMAIS le dialer du telephone. Il voit seulement un spinner "Paiement en cours..." puis "Paiement reussi !" ou "Echec".

Ce que voit le patient (UX)

  1. Il clique sur "Payer 5 000 XOF"
  2. Une modale s'ouvre : "Entrez votre code PIN MoMo" avec un champ masque (****)
  3. Il tape son PIN et appuie sur "Confirmer"
  4. Un spinner s'affiche : "Paiement en cours..."
  5. Apres 5-15 secondes : "Paiement reussi !" avec une coche verte
  6. Ses credits ou abonnement sont immediatement actifs

Ce qui se passe en arriere-plan (invisible)

  1. Le frontend envoie le PIN au plugin Capacitor natif Android
  2. Le plugin natif compose le code USSD en arriere-plan via TelephonyManager.sendUssdRequest()
  3. L'API Android execute l'appel USSD sans ouvrir le dialer
  4. La reponse USSD est interceptee programmatiquement
  5. Le resultat est renvoye au frontend via le bridge Capacitor
  6. Le backend est notifie et active le service

Stack Technique

  • Frontend : Next.js 14 (Capacitor WebView)
  • Plugin Natif : Plugin Capacitor Android custom en Java/Kotlin
  • API Android : TelephonyManager.sendUssdRequest() (API 26+, Android 8.0+)
  • Backend : NestJS + Prisma + PostgreSQL
  • Securite : PIN jamais stocke, jamais envoye au serveur, utilise uniquement dans l'appel USSD local

3.2 Plugin Principal

// android/app/src/main/java/com/tcardio/plugins/ussdpayment/UssdPaymentPlugin.java
package com.tcardio.plugins.ussdpayment;

import android.Manifest;
import android.content.Context;
import android.os.Build;
import android.os.Handler;
import android.os.Looper;
import android.telephony.TelephonyManager;
import android.telecom.TelecomManager;

import com.getcapacitor.JSObject;
import com.getcapacitor.Plugin;
import com.getcapacitor.PluginCall;
import com.getcapacitor.PluginMethod;
import com.getcapacitor.annotation.CapacitorPlugin;
import com.getcapacitor.annotation.Permission;

@CapacitorPlugin(
    name = "UssdPayment",
    permissions = {
        @Permission(
            alias = "phone",
            strings = { Manifest.permission.CALL_PHONE }
        )
    }
)
public class UssdPaymentPlugin extends Plugin {

    private UssdService ussdService;
    private SimDetector simDetector;

    @Override
    public void load() {
        Context context = getContext();
        this.simDetector = new SimDetector(context);
        this.ussdService = new UssdService(context);
    }

    // ---- Detecter l'operateur de la carte SIM ----
    @PluginMethod
    public void detectOperator(PluginCall call) {
        try {
            SimDetector.SimInfo simInfo = simDetector.detect();
            JSObject result = new JSObject();
            result.put("operator", simInfo.operator);         // ORANGE, MTN, MOOV, WAVE, FREE, UNKNOWN
            result.put("operatorName", simInfo.operatorName);  // "Orange CI", "MTN CI", etc.
            result.put("countryCode", simInfo.countryCode);    // ci, sn, ml, bf, cm
            result.put("phoneNumber", simInfo.phoneNumber);    // Peut etre null
            result.put("supportsUssd", simInfo.supportsUssd);  // true si Android 8.0+
            call.resolve(result);
        } catch (Exception e) {
            call.reject("Erreur detection operateur: " + e.getMessage());
        }
    }

    // ---- Executer le paiement USSD en arriere-plan ----
    @PluginMethod
    public void executePayment(PluginCall call) {
        // Verifier la permission CALL_PHONE
        if (!hasPermission(Manifest.permission.CALL_PHONE)) {
            requestPermissionForAlias("phone", call, "phonePermissionCallback");
            return;
        }

        // Verifier Android 8.0+ (API 26)
        if (Build.VERSION.SDK_INT < Build.VERSION_CODES.O) {
            call.reject("Android 8.0 ou superieur requis pour le paiement automatique");
            return;
        }

        // Recuperer les parametres
        String pin = call.getString("pin");
        int amount = call.getInt("amount", 0);
        String receiverNumber = call.getString("receiverNumber");
        String reference = call.getString("reference", "");
        String operator = call.getString("operator", "");   // Force un operateur specifique

        if (pin == null || pin.length() < 4 || pin.length() > 5) {
            call.reject("PIN invalide (4-5 chiffres requis)");
            return;
        }
        if (amount <= 0) {
            call.reject("Montant invalide");
            return;
        }
        if (receiverNumber == null || receiverNumber.isEmpty()) {
            call.reject("Numero recepteur manquant");
            return;
        }

        // Detecter l'operateur si pas force
        SimDetector.SimInfo simInfo = simDetector.detect();
        String detectedOperator = operator.isEmpty() ? simInfo.operator : operator;

        // Construire le code USSD
        String ussdCode = UssdCodeBuilder.build(
            detectedOperator,
            simInfo.countryCode,
            receiverNumber,
            amount,
            pin,
            reference
        );

        if (ussdCode == null) {
            call.reject("Operateur non supporte: " + detectedOperator);
            return;
        }

        // Executer l'appel USSD en arriere-plan
        ussdService.execute(ussdCode, new UssdResponseCallback() {
            @Override
            public void onSuccess(String ussdResponse) {
                JSObject result = new JSObject();
                result.put("success", true);
                result.put("operator", detectedOperator);
                result.put("amount", amount);
                result.put("reference", reference);
                // Parser la reponse USSD pour verifier le succes
                result.put("ussdResponse", sanitizeResponse(ussdResponse));
                result.put("confirmed", isPaymentConfirmed(ussdResponse, detectedOperator));
                call.resolve(result);
            }

            @Override
            public void onFailure(String errorCode, String errorMessage) {
                JSObject result = new JSObject();
                result.put("success", false);
                result.put("errorCode", errorCode);
                result.put("errorMessage", errorMessage);
                call.resolve(result);   // resolve pas reject, pour gerer cote JS
            }

            @Override
            public void onTimeout() {
                JSObject result = new JSObject();
                result.put("success", false);
                result.put("errorCode", "TIMEOUT");
                result.put("errorMessage", "Delai d'attente depasse (30s)");
                call.resolve(result);
            }
        });
    }

    // ---- Verifier si l'appareil supporte USSD auto ----
    @PluginMethod
    public void checkSupport(PluginCall call) {
        JSObject result = new JSObject();
        result.put("supported", Build.VERSION.SDK_INT >= Build.VERSION_CODES.O);
        result.put("androidVersion", Build.VERSION.SDK_INT);
        result.put("minRequired", Build.VERSION_CODES.O);  // 26
        result.put("hasCallPermission", hasPermission(Manifest.permission.CALL_PHONE));
        call.resolve(result);
    }

    @PluginMethod
    public void requestPhonePermission(PluginCall call) {
        requestPermissionForAlias("phone", call, "phonePermissionCallback");
    }

    @com.getcapacitor.annotation.PermissionCallback
    private void phonePermissionCallback(PluginCall call) {
        if (hasPermission(Manifest.permission.CALL_PHONE)) {
            JSObject result = new JSObject();
            result.put("granted", true);
            call.resolve(result);
        } else {
            call.reject("Permission CALL_PHONE refusee par l'utilisateur");
        }
    }

    // Nettoyer la reponse USSD (retirer les donnees sensibles)
    private String sanitizeResponse(String response) {
        if (response == null) return "";
        // Ne retourner que les mots cles succes/echec, pas les details du compte
        return response.replaceAll("[0-9]{5,}", "***");
    }

    // Verifier dans la reponse USSD si le paiement a reussi
    private boolean isPaymentConfirmed(String response, String operator) {
        if (response == null) return false;
        String lower = response.toLowerCase();
        // Mots cles de succes selon l'operateur
        return lower.contains("succes") ||
               lower.contains("effectue") ||
               lower.contains("confirme") ||
               lower.contains("reussi") ||
               lower.contains("successful") ||
               lower.contains("transaction id") ||
               lower.contains("merci");
    }
}

3.3 Service d'Execution USSD

// android/app/src/main/java/com/tcardio/plugins/ussdpayment/UssdService.java
package com.tcardio.plugins.ussdpayment;

import android.content.Context;
import android.net.Uri;
import android.os.Build;
import android.os.Handler;
import android.os.Looper;
import android.telephony.TelephonyManager;

import androidx.annotation.RequiresApi;

@RequiresApi(api = Build.VERSION_CODES.O)
public class UssdService {

    private final Context context;
    private final TelephonyManager telephonyManager;
    private static final int USSD_TIMEOUT_MS = 30000;  // 30 secondes

    public UssdService(Context context) {
        this.context = context;
        this.telephonyManager = (TelephonyManager)
            context.getSystemService(Context.TELEPHONY_SERVICE);
    }

    /**
     * Execute un code USSD en arriere-plan via TelephonyManager.
     * NE OUVRE PAS LE DIALER. L'appel est completement invisible.
     *
     * @param ussdCode  Le code USSD complet (ex: *144*1*1*0197548441*5000*REF*1234#)
     * @param callback  Callback pour le resultat
     */
    public void execute(String ussdCode, UssdResponseCallback callback) {
        Handler mainHandler = new Handler(Looper.getMainLooper());

        // Timer de timeout
        Runnable timeoutRunnable = () -> callback.onTimeout();
        mainHandler.postDelayed(timeoutRunnable, USSD_TIMEOUT_MS);

        try {
            // sendUssdRequest execute l'appel USSD en arriere-plan
            // SANS ouvrir le dialer du telephone
            telephonyManager.sendUssdRequest(
                ussdCode,
                new TelephonyManager.UssdResponseCallback() {
                    @Override
                    public void onReceiveUssdResponse(
                        TelephonyManager manager,
                        String request,
                        CharSequence response
                    ) {
                        // Annuler le timeout
                        mainHandler.removeCallbacks(timeoutRunnable);
                        // Succes : reponse USSD recue
                        callback.onSuccess(response.toString());
                    }

                    @Override
                    public void onReceiveUssdResponseFailed(
                        TelephonyManager manager,
                        String request,
                        int failureCode
                    ) {
                        // Annuler le timeout
                        mainHandler.removeCallbacks(timeoutRunnable);

                        String errorMessage;
                        String errorCode;
                        switch (failureCode) {
                            case TelephonyManager.USSD_ERROR_SERVICE_UNAVAIL:
                                errorCode = "SERVICE_UNAVAILABLE";
                                errorMessage = "Service USSD indisponible";
                                break;
                            case TelephonyManager.USSD_RETURN_FAILURE:
                                errorCode = "USSD_FAILURE";
                                errorMessage = "Echec de la requete USSD (PIN incorrect ou solde insuffisant)";
                                break;
                            default:
                                errorCode = "UNKNOWN_ERROR";
                                errorMessage = "Erreur USSD inconnue (code: " + failureCode + ")";
                        }
                        callback.onFailure(errorCode, errorMessage);
                    }
                },
                mainHandler
            );
        } catch (SecurityException e) {
            mainHandler.removeCallbacks(timeoutRunnable);
            callback.onFailure("PERMISSION_DENIED", "Permission CALL_PHONE non accordee");
        } catch (Exception e) {
            mainHandler.removeCallbacks(timeoutRunnable);
            callback.onFailure("EXECUTION_ERROR", e.getMessage());
        }
    }
}

// android/app/src/main/java/com/tcardio/plugins/ussdpayment/UssdCodeBuilder.java
package com.tcardio.plugins.ussdpayment;

/**
 * Construit le code USSD complet selon l'operateur et le pays.
 * Le PIN est insere directement dans le code USSD pour paiement automatique.
 * Le numero recepteur est configure sur le serveur, le patient ne le voit JAMAIS.
 */
public class UssdCodeBuilder {

    /**
     * Construit le code USSD de transfert d'argent.
     *
     * @param operator        ORANGE, MTN, MOOV, WAVE, FREE
     * @param countryCode     ci, sn, ml, bf, cm, tg, gn, bj
     * @param receiverNumber  Numero du compte recepteur (de l'application)
     * @param amount          Montant en XOF (entier, multiple de 5)
     * @param pin             Code PIN du client (4-5 chiffres)
     * @param reference       Reference de la transaction (optionnel)
     * @return Code USSD complet ou null si operateur non supporte
     */
    public static String build(
        String operator,
        String countryCode,
        String receiverNumber,
        int amount,
        String pin,
        String reference
    ) {
        if (operator == null) return null;

        switch (operator.toUpperCase()) {

            // ===== ORANGE MONEY =====
            case "ORANGE":
                return buildOrangeMoney(countryCode, receiverNumber, amount, pin);

            // ===== MTN MOMO =====
            case "MTN":
                return buildMtnMomo(countryCode, receiverNumber, amount, pin);

            // ===== MOOV MONEY =====
            case "MOOV":
                return buildMoovMoney(countryCode, receiverNumber, amount, pin);

            // ===== FREE MONEY (Senegal) =====
            case "FREE":
                return buildFreeMoney(receiverNumber, amount, pin);

            default:
                return null;
        }
    }

    // ----- Orange Money -----
    private static String buildOrangeMoney(String country, String receiver, int amount, String pin) {
        switch (country.toLowerCase()) {
            case "ci":  // Cote d'Ivoire
                // Format: #144*1*1*NUMERO*MONTANT*PIN#
                return "#144*1*1*" + receiver + "*" + amount + "*" + pin + "#";

            case "sn":  // Senegal
                // Format: #144*1*NUMERO*MONTANT*PIN#
                return "#144*1*" + receiver + "*" + amount + "*" + pin + "#";

            case "ml":  // Mali
                // Format: #144*1*1*NUMERO*MONTANT*PIN#
                return "#144*1*1*" + receiver + "*" + amount + "*" + pin + "#";

            case "bf":  // Burkina Faso
                // Format: #144*1*1*NUMERO*MONTANT*PIN#
                return "#144*1*1*" + receiver + "*" + amount + "*" + pin + "#";

            case "cm":  // Cameroun
                // Format: #150*1*1*NUMERO*MONTANT*PIN#
                return "#150*1*1*" + receiver + "*" + amount + "*" + pin + "#";

            case "gn":  // Guinee
                // Format: #144*1*1*NUMERO*MONTANT*PIN#
                return "#144*1*1*" + receiver + "*" + amount + "*" + pin + "#";

            default:
                return "#144*1*1*" + receiver + "*" + amount + "*" + pin + "#";
        }
    }

    // ----- MTN MoMo -----
    private static String buildMtnMomo(String country, String receiver, int amount, String pin) {
        switch (country.toLowerCase()) {
            case "ci":  // Cote d'Ivoire
                // Format: *133*1*1*NUMERO*MONTANT*PIN#
                return "*133*1*1*" + receiver + "*" + amount + "*" + pin + "#";

            case "cm":  // Cameroun
                // Format: *126*1*NUMERO*MONTANT*PIN#
                return "*126*1*" + receiver + "*" + amount + "*" + pin + "#";

            case "bj":  // Benin
                // Format: *880*1*NUMERO*MONTANT*PIN#
                return "*880*1*" + receiver + "*" + amount + "*" + pin + "#";

            case "gn":  // Guinee
                // Format: *133*1*1*NUMERO*MONTANT*PIN#
                return "*133*1*1*" + receiver + "*" + amount + "*" + pin + "#";

            default:
                return "*133*1*1*" + receiver + "*" + amount + "*" + pin + "#";
        }
    }

    // ----- Moov Money -----
    private static String buildMoovMoney(String country, String receiver, int amount, String pin) {
        switch (country.toLowerCase()) {
            case "ci":  // Cote d'Ivoire
                // Format: *155*1*1*NUMERO*MONTANT*PIN#
                return "*155*1*1*" + receiver + "*" + amount + "*" + pin + "#";

            case "bf":  // Burkina Faso
                // Format: *555*1*1*NUMERO*MONTANT*PIN#
                return "*555*1*1*" + receiver + "*" + amount + "*" + pin + "#";

            case "tg":  // Togo
                // Format: *155*1*NUMERO*MONTANT*PIN#
                return "*155*1*" + receiver + "*" + amount + "*" + pin + "#";

            case "bj":  // Benin
                // Format: *155*1*NUMERO*MONTANT*PIN#

3.6 Detection Automatique de l'Operateur SIM

// android/app/src/main/java/com/tcardio/plugins/ussdpayment/SimDetector.java
package com.tcardio.plugins.ussdpayment;

import android.content.Context;
import android.telephony.TelephonyManager;

/**
 * Detecte automatiquement l'operateur de la carte SIM inseree.
 * Utilise le MCC/MNC (Mobile Country Code / Mobile Network Code).
 */
public class SimDetector {

    private final TelephonyManager telephonyManager;

    public SimDetector(Context context) {
        this.telephonyManager = (TelephonyManager)
            context.getSystemService(Context.TELEPHONY_SERVICE);
    }

    public SimInfo detect() {
        String simOperator = telephonyManager.getSimOperator();    // MCC+MNC (ex: "61203")
        String simOperatorName = telephonyManager.getSimOperatorName();
        String simCountry = telephonyManager.getSimCountryIso();   // "ci", "sn", etc.

        SimInfo info = new SimInfo();
        info.countryCode = simCountry != null ? simCountry.toLowerCase() : "ci";
        info.operatorName = simOperatorName != null ? simOperatorName : "Inconnu";
        info.supportsUssd = android.os.Build.VERSION.SDK_INT >= 26;

        // Recuperer numero si disponible
        try {
            info.phoneNumber = telephonyManager.getLine1Number();
        } catch (SecurityException e) {
            info.phoneNumber = null;
        }

        // Identifier l'operateur par MCC/MNC
        if (simOperator != null && simOperator.length() >= 5) {
            info.operator = identifyByMccMnc(simOperator, info.countryCode);
        } else {
            // Fallback : identifier par nom
            info.operator = identifyByName(simOperatorName);
        }

        return info;
    }

    private String identifyByMccMnc(String mccmnc, String country) {
        // MCC 612 = Cote d'Ivoire
        // MCC 608 = Senegal
        // MCC 610 = Mali (corrige)
        // MCC 613 = Burkina Faso
        // MCC 624 = Cameroun
        // MCC 615 = Togo
        // MCC 611 = Guinee
        // MCC 616 = Benin

        switch (mccmnc) {
            // === COTE D'IVOIRE (612) ===
            case "61203": return "ORANGE";       // Orange CI
            case "61205": return "MTN";          // MTN CI
            case "61202": return "MOOV";         // Moov CI
            case "61206": return "ORANGE";       // Orange CI alt

            // === SENEGAL (608) ===
            case "60801": return "ORANGE";       // Orange SN (Sonatel)
            case "60802": return "FREE";         // Free SN (Tigo)
            case "60803": return "ORANGE";       // Expresso -> fallback Orange

            // === MALI (610) ===
            case "61001": return "ORANGE";       // Orange Mali (Malitel)
            case "61002": return "MOOV";         // Moov Mali

            // === BURKINA FASO (613) ===
            case "61301": return "ORANGE";       // Orange BF (ONATEL)
            case "61302": return "MOOV";         // Moov BF

            // === CAMEROUN (624) ===
            case "62401": return "MTN";          // MTN CM
            case "62402": return "ORANGE";       // Orange CM

            // === TOGO (615) ===
            case "61501": return "MOOV";         // Moov TG (Togo Cellulaire)
            case "61503": return "MOOV";         // Moov TG

            // === GUINEE (611) ===
            case "61101": return "ORANGE";       // Orange GN (Cellcom -> Orange)
            case "61102": return "MTN";          // MTN GN (Areeba)

            // === BENIN (616) ===
            case "61601": return "MTN";          // MTN BJ (Libercom)
            case "61602": return "MOOV";         // Moov BJ
            case "61604": return "MTN";          // MTN BJ

            default: return "UNKNOWN";
        }
    }

    private String identifyByName(String name) {
        if (name == null) return "UNKNOWN";
        String lower = name.toLowerCase();
        if (lower.contains("orange")) return "ORANGE";
        if (lower.contains("mtn"))    return "MTN";
        if (lower.contains("moov"))   return "MOOV";
        if (lower.contains("wave"))   return "WAVE";
        if (lower.contains("free") || lower.contains("tigo")) return "FREE";
        if (lower.contains("airtel")) return "AIRTEL";
        return "UNKNOWN";
    }

    public static class SimInfo {
        public String operator;       // ORANGE, MTN, MOOV, WAVE, FREE, UNKNOWN
        public String operatorName;   // Nom complet ex: "Orange CI"
        public String countryCode;    // ci, sn, ml, bf, cm, tg, gn, bj

// frontend-web/lib/ussd-payment.ts
import { registerPlugin } from '@capacitor/core';

// Interface TypeScript du plugin natif
interface UssdPaymentPlugin {
  detectOperator(): Promise<{
    operator: string;       // ORANGE, MTN, MOOV, FREE, UNKNOWN
    operatorName: string;
    countryCode: string;
    phoneNumber: string | null;
    supportsUssd: boolean;
  }>;

  executePayment(options: {
    pin: string;
    amount: number;
    receiverNumber: string;
    reference: string;
    operator?: string;       // Forcer un operateur
  }): Promise<{
    success: boolean;
    operator: string;
    amount: number;
    reference: string;
    ussdResponse?: string;
    confirmed?: boolean;      // True si reponse USSD indique succes
    errorCode?: string;
    errorMessage?: string;
  }>;

  checkSupport(): Promise<{
    supported: boolean;
    androidVersion: number;
    minRequired: number;
    hasCallPermission: boolean;
  }>;

  requestPhonePermission(): Promise<{
    granted: boolean;
  }>;
}

// Enregistrer le plugin
const UssdPayment = registerPlugin<UssdPaymentPlugin>('UssdPayment');

export default UssdPayment;

// ===== Helper pour le paiement complet =====
export async function processUssdPayment(params: {
  pin: string;
  amount: number;
  receiverNumber: string;
  reference: string;
}): Promise<{
  success: boolean;
  confirmed: boolean;
  operator: string;
  errorMessage?: string;
}> {
  // 1. Verifier le support
  const support = await UssdPayment.checkSupport();
  if (!support.supported) {
    return {
      success: false,
      confirmed: false,
      operator: 'UNKNOWN',
      errorMessage: 'Votre telephone ne supporte pas le paiement automatique (Android 8.0+ requis)',
    };
  }

  // 2. Demander la permission si necessaire
  if (!support.hasCallPermission) {
    try {
      await UssdPayment.requestPhonePermission();
    } catch {
      return {
        success: false,
        confirmed: false,
        operator: 'UNKNOWN',
        errorMessage: 'Permission telephonique refusee. Activez-la dans les parametres.',
      };
    }
  }

  // 3. Executer le paiement
  const result = await UssdPayment.executePayment(params);

  return {
    success: result.success,
    confirmed: result.confirmed || false,
    operator: result.operator,
    errorMessage: result.errorMessage,
  };
}

// frontend-web/components/payment/PinInput.tsx
'use client';

import { useState, useRef, useEffect } from 'react';

interface PinInputProps {
  length?: number;          // 4 ou 5 chiffres
  onComplete: (pin: string) => void;
  disabled?: boolean;
}

export function PinInput({ length = 4, onComplete, disabled }: PinInputProps) {
  const [digits, setDigits] = useState<string[]>(Array(length).fill(''));
  const inputRefs = useRef<(HTMLInputElement | null)[]>([]);

  const handleChange = (index: number, value: string) => {
    if (!/^\d*$/.test(value)) return;  // Chiffres uniquement

    const newDigits = [...digits];
    newDigits[index] = value.slice(-1);   // Un seul chiffre
    setDigits(newDigits);

    // Auto-focus champ suivant
    if (value && index < length - 1) {
      inputRefs.current[index + 1]?.focus();
    }

    // Verifier si complet
    const pin = newDigits.join('');
    if (pin.length === length && newDigits.every(d => d !== '')) {
      onComplete(pin);
    }
  };

  const handleKeyDown = (index: number, e: React.KeyboardEvent) => {
    if (e.key === 'Backspace' && !digits[index] && index > 0) {
      inputRefs.current[index - 1]?.focus();
    }
  };

  return (
    <div className="flex gap-3 justify-center my-6">
      {digits.map((digit, i) => (
        <input
          key={i}
          ref={(el) => { inputRefs.current[i] = el; }}
          type="password"
          inputMode="numeric"
          maxLength={1}
          value={digit}
          onChange={(e) => handleChange(i, e.target.value)}
          onKeyDown={(e) => handleKeyDown(i, e)}
          disabled={disabled}
          className="w-14 h-16 text-center text-2xl font-bold
                     bg-gray-800 border-2 border-gray-600 rounded-xl
                     text-white focus:border-amber-500 focus:outline-none
                     disabled:opacity-50"
          autoComplete="off"
        />
      ))}
    </div>
  );
}

type PaymentStep = 'pin' | 'processing' | 'success' | 'error';

export function PaymentModal({
  isOpen, onClose, amount, description, packageId, type, onSuccess
}: PaymentModalProps) {
  const [step, setStep] = useState<PaymentStep>('pin');
  const [operator, setOperator] = useState<string>('');
  const [operatorName, setOperatorName] = useState<string>('');
  const [errorMessage, setErrorMessage] = useState('');
  const [supported, setSupported] = useState(true);

  // Detecter l'operateur au montage
  useEffect(() => {
    if (isOpen) {
      setStep('pin');
      setErrorMessage('');
      detectOperator();
    }
  }, [isOpen]);

  const detectOperator = async () => {
    try {
      const info = await UssdPayment.detectOperator();
      setOperator(info.operator);
      setOperatorName(info.operatorName);
      setSupported(info.supportsUssd);
    } catch {
      setSupported(false);
    }
  };

  const handlePinComplete = async (pin: string) => {
    setStep('processing');

    try {
      // 1. Creer le paiement cote serveur (statut PENDING)
      const { data: payment } = await api.post('/payments/ussd/initiate', {
        type,
        packageId,
        amount,
        operator,
      });

      // 2. Executer l'appel USSD en arriere-plan
      // Le PIN reste sur l'appareil, JAMAIS envoye au serveur
      const result = await processUssdPayment({
        pin,
        amount,
        receiverNumber: payment.receiverNumber,   // Fourni par le serveur
        reference: payment.reference,
      });

      if (result.success && result.confirmed) {
        // 3. Confirmer le paiement au serveur
        await api.post(`/payments/ussd/${payment.paymentId}/confirm`, {
          operator: result.operator,
          ussdConfirmed: true,
        });

        setStep('success');
        setTimeout(() => {
          onSuccess();
          onClose();
        }, 2000);

      } else {
        // Paiement echoue
        setErrorMessage(
          result.errorMessage || 'Paiement echoue. Verifiez votre PIN et votre solde.'
        );
        setStep('error');

        // Notifier le serveur de l'echec
        await api.post(`/payments/ussd/${payment.paymentId}/failed`, {
          errorCode: 'USSD_FAILED',
          errorMessage: result.errorMessage,
        });
      }

    } catch (error: any) {
      setErrorMessage(error.message || 'Erreur inattendue');
      setStep('error');
    }
  };

  if (!isOpen) return null;

  return (
    <div className="fixed inset-0 z-50 flex items-center justify-center bg-black/70">

          </p>

            <span className="inline-block bg-amber-500/20 text-amber-300
                            px-3 py-1 rounded-full text-sm mt-2">
              {operatorName}
            </span>
          )}
        </div>

        {/* Etape: Saisie PIN */}
        {step === 'pin' && (

              Entrez votre code PIN Mobile Money
            </p>
            <PinInput length={4} onComplete={handlePinComplete} />

            <button onClick={onClose}

              Annuler
            </button>
          </div>
        )}

        {/* Etape: Traitement en cours */}
        {step === 'processing' && (
          <div className="text-center py-8">
            <div className="w-16 h-16 border-4 border-amber-500 border-t-transparent

// backend/src/modules/payment/payment.service.ts

@Injectable()
export class PaymentService {

  async createUssdPayment(patientId: string, dto: InitiateUssdDto) {
    const pkg = this.getPackageOrPlan(dto.packageId, dto.type);
    const reference = this.generateReference();

    const payment = await this.prisma.payment.create({
      data: {
        patientId,
        type: dto.type,
        amount: pkg.price,
        currency: 'XOF',
        status: 'PENDING',
        provider: 'MOMO_LOCAL',
        description: `Paiement ${pkg.name}`,
        momoReference: reference,
        momoReceiverNumber: this.config.get('payment.momoReceiverNumber'),
        metadata: { packageId: dto.packageId, operator: dto.operator },
        expiresAt: new Date(Date.now() + 5 * 60 * 1000), // 5 min pour USSD auto
      },
    });

    // Log evenement
    await this.logEvent(payment.id, 'USSD_INITIATED', null, 'PENDING');

    return payment;
  }

  async confirmUssdPayment(paymentId: string, patientId: string, dto: ConfirmUssdDto) {
    // Verrou Redis pour eviter double traitement
    const lock = await this.redis.set(`payment:confirm:${paymentId}`, '1', 'EX', 30, 'NX');
    if (!lock) throw new ConflictException('Paiement deja en cours de traitement');

    try {
      const payment = await this.prisma.payment.findFirstOrThrow({
        where: { id: paymentId, patientId, status: 'PENDING' },
      });

      await this.prisma.$transaction(async (tx) => {
        // 1. Mettre a jour le paiement
        await tx.payment.update({
          where: { id: paymentId },
          data: {
            status: 'PROCESSING',   // En attente de verification admin/auto
            channel: this.mapOperator(dto.operator),
            completedAt: new Date(),
          },
        });

        // 2. Logger
        await this.logEvent(paymentId, 'USSD_CONFIRMED_BY_CLIENT', 'PENDING', 'PROCESSING');

        // 3. Option A: Auto-validation (si on fait confiance a la reponse USSD)
        if (dto.ussdConfirmed) {
          await tx.payment.update({
            where: { id: paymentId },
            data: { status: 'COMPLETED' },
          });

          // Activer abonnement ou credits
          const meta = payment.metadata as any;
          if (payment.type === 'SUBSCRIPTION') {
            await this.subscriptionService.activate(patientId, meta.packageId, paymentId, payment.amount);
          } else {
            const pkg = this.getPackage(meta.packageId);
            await this.creditService.addCredits(patientId, pkg.credits + pkg.bonus, paymentId);
          }

          await this.logEvent(paymentId, 'PAYMENT_COMPLETED', 'PROCESSING', 'COMPLETED');
          this.notificationGateway.sendToUser(patientId, 'payment:completed', {
            paymentId, amount: payment.amount,
          });
        }
        // Option B: Attente validation admin (voir section admin dashboard)
      });

      return { success: true, status: dto.ussdConfirmed ? 'COMPLETED' : 'PROCESSING' };

    } finally {
      await this.redis.del(`payment:confirm:${paymentId}`);
    }
  }

  async failUssdPayment(paymentId: string, patientId: string, dto: any) {
    await this.prisma.payment.update({
      where: { id: paymentId },
      data: {
        status: 'FAILED',
        failedAt: new Date(),
        providerRawData: dto,
      },
    });
    await this.logEvent(paymentId, 'USSD_FAILED', 'PENDING', 'FAILED');
    return { success: true };
  }

  private generateReference(): string {
    const ts = Date.now().toString(36).toUpperCase().slice(-4);
    const rand = Math.random().toString(36).substring(2, 6).toUpperCase();
    return `TC-${ts}-${rand}`;
  }
}

9. Securite

REGLES DE SECURITE CRITIQUES

Protection du PIN

  • Le PIN MoMo est saisi dans le frontend et envoye UNIQUEMENT au plugin natif Android local
  • Le PIN n'est JAMAIS envoye au serveur backend via l'API
  • Le PIN n'est JAMAIS stocke en memoire apres l'appel USSD
  • Le PIN n'est JAMAIS ecrit dans les logs (ni frontend, ni backend)
  • Le PIN est utilise une seule fois : dans UssdCodeBuilder.build() pour construire le code USSD
  • Apres l'appel sendUssdRequest(), la variable PIN est mise a null

Protection du Numero Recepteur

  • Le numero recepteur est stocke uniquement dans les variables d'environnement du serveur
  • Il est transmis au frontend UNIQUEMENT lors de l'initiation du paiement (reponse API)
  • Il n'est JAMAIS affiche dans l'interface utilisateur
  • Le frontend le passe au plugin natif qui construit le code USSD en arriere-plan
  • Le patient ne voit que : montant + champ PIN + bouton confirmer

Anti-Fraude

  • Maximum 3 tentatives de paiement echouees par patient par heure
  • Verrou Redis pour empecher les doubles confirmations
  • Expiration automatique des paiements PENDING apres 5 minutes
  • Log de tous les evenements de paiement (PaymentEvent)
  • Verification admin optionnelle pour les gros montants (> 25 000 XOF)
  • Alerte admin en cas de 5+ echecs consecutifs pour un patient
// Anti-fraude : limiter les tentatives
async checkRateLimit(patientId: string): Promise<boolean> {
  const key = `payment:attempts:${patientId}`;
  const attempts = await this.redis.incr(key);
  if (attempts === 1) {
    await this.redis.expire(key, 3600); // Reset apres 1 heure
  }
  return attempts <= 3; // Max 3 par heure
}

12. Resume du Flux Complet

1
Patient ouvre la page Paiement - Choisit un plan ou pack de credits, clique "Payer"
2
Modale s'ouvre - Affiche montant, operateur detecte automatiquement (Orange, MTN, etc.), champ PIN masque
3
Patient saisit son PIN - 4 ou 5 chiffres, champs individuels avec auto-focus
4
Frontend appelle POST /payments/ussd/initiate - Backend cree un paiement PENDING, retourne le numero recepteur et la reference. Le numero recepteur n'est PAS affiche.
5
Plugin natif construit le code USSD - UssdCodeBuilder.build(operator, country, receiver, amount, pin, ref) - Le code USSD contient le PIN et le numero recepteur mais n'est JAMAIS affiche.
6
TelephonyManager.sendUssdRequest() - L'appel USSD s'execute en arriere-plan. Pas de dialer. Pas de popup. Invisible.
7
Reponse USSD interceptee - Le plugin analyse la reponse pour detecter "succes" ou "echec"
8
Si succes - Frontend appelle POST /payments/ussd/{id}/confirm. Backend active l'abonnement ou ajoute les credits. Notification temps reel.
9
Si echec - Frontend affiche l'erreur. Bouton "Reessayer". Le patient peut re-saisir son PIN.
10
Patient voit "Paiement reussi !" - Coche verte, service active. Tout le processus prend 5-15 secondes.
