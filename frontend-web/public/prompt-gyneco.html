T-Gyneco Plus

Prompt complet pour Claude Code - Application de telemedecine gynecologique

1. Presentation du Projet

Nom : T-Gyneco Plus

Description : Plateforme de telemedecine gynecologique complete permettant le suivi du cycle menstruel, de la grossesse, de la fertilite, de la contraception, des depistages et de la menopause. L'application connecte les patientes avec leurs gynecologues et sages-femmes pour un suivi medical continu, avec analyse IA des donnees, teleconsultation video, messagerie, ordonnances electroniques et paiement Mobile Money.

Marche cible : Afrique de l'Ouest francophone (Benin, Togo, Cote d'Ivoire, Senegal)

Langue : Francais (sans accents dans le code, avec accents dans l'UI)

Monnaie : XOF (Franc CFA)

2. Stack Technique

Backend

  • NestJS 10 - Framework API REST
  • TypeScript - Langage
  • Prisma ORM 5 - ORM PostgreSQL
  • PostgreSQL 15 - Base de donnees
  • Redis 7 - Cache & temps reel
  • MinIO - Stockage objets (fichiers, rapports, echographies)
  • Socket.IO - Communication temps reel
  • DeepSeek API - Analyse IA gynecologique
  • OpenAI Vision - OCR documents medicaux
  • JWT - Authentification avec refresh tokens
  • pdf-lib - Generation PDF
  • web-push - Notifications push
  • nodemailer - Emails

Frontend Web

  • Next.js 14 - Framework React (App Router)
  • React 18 - UI
  • Tailwind CSS - Styling (theme dark mauve/rose)
  • TanStack Query - Data fetching & cache
  • Zustand - State management
  • Chart.js - Graphiques (cycles, temperature, poids)
  • React Hook Form + Zod - Formulaires
  • Socket.IO Client - Temps reel
  • Axios - Client HTTP
  • Lucide React - Icones
  • date-fns - Manipulation dates

Mobile

  • Capacitor 8 - Wrapper Android APK
  • WebView - Charge le site web

DevOps & Deploiement

  • Docker + Docker Compose - Conteneurisation
  • Nginx - Reverse proxy + SSL
  • Certbot - Certificats Let's Encrypt
  • VPS Linux (Ubuntu 20+)

4. Roles Utilisateurs

RoleDescriptionPermissions
PATIENTEFemme utilisant l'application pour son suivi gynecologiqueSaisir cycles, suivi grossesse, fertilite, contraception, teleconsultation, messagerie, rapports
MEDECINMedecin generaliste avec competences gynecologiquesVoir patientes liees, teleconsultation, prescriptions, rapports, notes medicales
GYNECOLOGUESpecialiste en gynecologie-obstetriqueMemes droits que MEDECIN + suivi grossesse avance, echographies, chirurgie
SAGE_FEMMESage-femme (suivi grossesse et accouchement)Suivi grossesse, accouchement, post-partum, prescriptions limitees
ADMINAdministrateur systemeGestion utilisateurs, verification medecins, paiements, configuration IA, audit

6. Fonctionnalites Detaillees

6.1 Suivi du Cycle Menstruel

  • Saisie du premier et dernier jour des regles
  • Intensite du flux (spotting, leger, moyen, abondant, tres abondant)
  • Symptomes multiples (crampes, maux de tete, fatigue, ballonnements, acne, sensibilite seins, etc.)
  • Niveau de douleur (echelle 0-10)
  • Prediction du prochain cycle base sur l'historique (moyenne glissante)
  • Detection de cycles irreguliers (alerte IA si ecart > 7 jours)
  • Graphique d'evolution de la duree des cycles
  • Phase actuelle du cycle (menstruation, folliculaire, ovulation, luteale)
  • Historique complet avec pagination
  • Notes libres par cycle

6.2 Suivi de Grossesse

  • Creation de grossesse avec date des dernieres regles (DDR)
  • Calcul automatique de la DPA (date prevue d'accouchement = DDR + 280 jours)
  • Semaine de grossesse actuelle (SA) calculee automatiquement
  • Suivi du poids semaine par semaine avec graphique de courbe de poids
  • Suivi de la tension arterielle pendant la grossesse
  • Calendrier des rendez-vous prenataux (consultations, echographies, prises de sang, monitoring)
  • 3 echographies obligatoires : T1 (12 SA), T2 (22 SA), T3 (32 SA)
  • Upload et stockage des images d'echographie (MinIO)
  • Mesures foetales (LCC, BIP, PC, PA, LF) en JSON
  • Suivi des complications (diabete gestationnel, pre-eclampsie, etc.)
  • Fin de grossesse : accouchement (voie basse/cesarienne), poids bebe, sexe
  • Compteur de grossesses (gravida) et accouchements (para)

6.3 Calendrier de Fertilite

  • Calcul de la fenetre fertile basee sur l'historique des cycles
  • Date d'ovulation estimee (longueur du cycle - 14 jours)
  • Suivi de la temperature basale (courbe de temperature)
  • Methodes de mesure : buccale, vaginale, rectale
  • Detection du pic de temperature post-ovulatoire
  • Calendrier visuel avec jours fertiles colores
  • Predictions basees sur les 6 derniers cycles minimum
  • Alertes de fenetre fertile (notification push)

6.4 Suivi Contraception

  • Type de contraception (pilule, sterilet, implant, patch, anneau, preservatif, etc.)
  • Marque et dosage du produit
  • Date de debut et rappel de renouvellement
  • Rappel quotidien pour la pilule (heure configurable)
  • Suivi des effets secondaires
  • Score de satisfaction (1-5)
  • Historique des changements de contraception

6.5 Depistages & Examens

  • Types : frottis cervical, mammographie, echographie pelvienne/mammaire, densitometrie osseuse, bilan hormonal, test HPV, colposcopie, biopsie
  • Resultats : normal, anormal, en attente, a surveiller, necessite suivi
  • Upload de documents/resultats (PDF, images) sur MinIO
  • Rappels automatiques selon les recommandations (frottis tous les 3 ans, mammographie tous les 2 ans apres 50 ans)
  • Historique complet avec filtres par type
  • Alerte quand un depistage est du

6.6 Suivi Menopause

  • Journal des symptomes (bouffees de chaleur, sueurs nocturnes, secheresse, troubles sommeil, etc.)
  • Severite des symptomes (echelle 1-10)
  • Suivi des traitements (THS, phytotherapie, complements)
  • Graphique d'evolution des symptomes dans le temps
  • Detecte automatiquement la menopause si absence de regles > 12 mois

6.7 Analyse IA (DeepSeek)

  • Analyse des donnees de cycle (regularite, duree, symptomes recurrents)
  • Analyse de la grossesse (prise de poids, tension, risques)
  • Analyse de fertilite (patterns de temperature, predictions)
  • Resume pour la patiente (langage simple) et pour le medecin (langage medical)
  • Niveau de risque : FAIBLE, MODERE, ELEVE, CRITIQUE
  • Score de confiance de l'analyse
  • Recommandations personnalisees
  • Filtrage des recommandations medicamenteuses (l'IA ne prescrit pas)
  • Historique des analyses avec pagination

6.8 Teleconsultation Video

  • Planification avec creneaux disponibles du medecin
  • Disponibilites hebdomadaires configurables par le medecin
  • Periodes d'indisponibilite (conges, absences)
  • Video WebRTC avec signaling Socket.IO
  • Chat textuel pendant la consultation
  • Envoi de fichiers (max 10 MB)
  • Resume de fin de consultation
  • Cout : 5000 credits (configurable)

6.9 Messagerie Directe

  • Conversations patiente-medecin persistantes
  • Messages texte + pieces jointes
  • Statut de lecture avec horodatage
  • Compteur de messages non lus

6.10 Ordonnances Electroniques

  • Creation par le medecin (medicaments, posologie, duree)
  • Generation PDF avec signature electronique et numero RPPS
  • Telechargement par la patiente
  • Lien optionnel vers une teleconsultation

6.11 Rapports PDF

  • Types : hebdomadaire, mensuel, trimestriel, personnalise, urgence, grossesse
  • Contenu : statistiques, graphiques, analyses IA, notes medicales
  • Signature electronique du medecin
  • Stockage sur MinIO, telechargement avec token JWT
  • Compteur de telechargements

6.12 Paiement Mobile Money (MoMo)

  • Packs credits :
    • Essentiel : 5 000 XOF → 5 000 credits
    • Standard : 10 000 XOF → 10 500 credits (+500 bonus)
    • Premium : 25 000 XOF → 27 500 credits (+2 500 bonus)
    • Mega : 50 000 XOF → 57 000 credits (+7 000 bonus)
  • Abonnements : BASIC (2 000 XOF/an), PRO (5 000 XOF/an)
  • Code USSD genere automatiquement pour MTN MoMo
  • Validation manuelle par l'admin
  • Historique des paiements et transactions de credits

6.13 Chatbot IA

  • Assistant gynecologique alimente par DeepSeek
  • Questions sur les cycles, grossesse, contraception, menopause
  • Contexte patient (historique des mesures)
  • Ne donne jamais de prescription ni de diagnostic
  • Historique des conversations

6.14 Notifications

  • Push notifications (web-push, service worker)
  • Rappels de prise de pilule, rendez-vous, depistages
  • Alertes IA (risque eleve detecte)
  • Notifications de teleconsultation (planifiee, demarree)
  • Notifications admin (messages systeme)
  • Gestion lu/non lu

6.15 Panel Admin

  • Dashboard avec statistiques globales (patientes, medecins, teleconsultations, revenus)
  • Gestion des utilisateurs (liste, recherche, suspension, suppression)
  • Verification des medecins/gynecologues (approuver/rejeter)
  • Validation des paiements MoMo
  • Configuration des seuils IA
  • Envoi de messages aux utilisateurs
  • Logs d'audit complets
  • Attribution de credits bonus et abonnements gratuits

  6. Phase 6 — IA : Integration DeepSeek (analyse cycles, grossesse, fertilite), chatbot IA, analyse de risque
  7. Phase 7 — Communication : Teleconsultation (scheduling, WebRTC, chat), messagerie directe, notifications push
  8. Phase 8 — Documents : Rapports PDF (generation, signature, download), ordonnances electroniques
  9. Phase 9 — Paiement : Credits, abonnements, paiement MoMo (USSD), validation admin
  10. Phase 10 — Admin : Dashboard admin, gestion utilisateurs, verification medecins, configuration IA, audit logs
  11. Phase 11 — Frontend complet : Toutes les pages Next.js avec design dark mauve, responsive mobile-first, graphiques
  12. Phase 12 — Deploiement : Docker Compose production, Nginx, SSL Let's Encrypt, Capacitor APK Android

  • TypeScript strict partout
  • Pas d'accents dans le code, uniquement dans les strings UI

  • Gestion d'erreurs robuste avec try/catch
  • Rate limiting (100 req/min global, 10 req/min auth)
